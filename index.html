Section 2

How the service works

A user sends a message to Lumi on WhatsApp. The conversation is initiated by the user. Lumi responds using AI-generated messages. The user may stop the conversation at any time. All conversations are private and user-controlled.

Flow

  • User sends a message to Lumi on WhatsApp
  • Conversation is initiated by the user
  • Lumi replies with AI-generated messages
  • User can stop at any time

What the service does not do

  • Send unsolicited messages
  • Perform cold outreach
  • Send marketing broadcasts
  • Add users to groups
